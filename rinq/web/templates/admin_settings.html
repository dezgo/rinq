Google Drive Storage

Call recordings are uploaded to Google Drive for 12-month retention. You must create the folder manually and paste its ID here.

{% if settings.get('drive_recordings_folder_id', {}).get('value') %} Configured {% else %} Not configured {% endif %}
Setup Steps:
  1. Enable the Google Drive API
    In Google Cloud Console, ensure the Drive API is enabled for your project.
  2. Add Drive scope to domain-wide delegation
    In Google Workspace Admin → Security → API Controls → Domain-wide Delegation, find the service account and add this scope:
    https://www.googleapis.com/auth/drive.file
  3. Create a folder in Google Drive
    Go to Google Drive and create a new folder (e.g., "{{ product_name }} Recordings"). Copy the folder ID from the URL: drive.google.com/drive/folders/FOLDER_ID
  4. Share the folder with the service account
    Right-click the folder → Share → Add fred-bot@bot-team-478101.iam.gserviceaccount.com with Editor access.
  5. Paste the folder ID above and save

⚠️ Note: After making changes in Google Cloud Console or Admin Console, it may take a few minutes for changes to propagate. Restart {{ product_name }} after configuration.
